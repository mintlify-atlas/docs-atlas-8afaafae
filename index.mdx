Antdv Next is a Vue 3 enterprise UI component library that brings the full power of [Ant Design](https://ant.design/) to the Vue ecosystem. It ships with 70+ battle-tested components, complete TypeScript definitions, powerful CSS-in-JS theming, and internationalization support for over 70 languages — everything you need to build production-grade web applications.

## Why Antdv Next?

Antdv Next is a faithful Vue 3 port of Ant Design, adapted for idiomatic Vue 3 Composition API with TSX. Every component ships with typed `Props`, `Emits`, and `SlotsType`, making autocomplete and type checking work out of the box.

## Register globally

```typescript main.ts
import { createApp } from 'vue'
import AntdvNext from 'antdv-next'
import 'antdv-next/dist/reset.css'
import App from './App.vue'

const app = createApp(App)
app.use(AntdvNext)
app.mount('#app')
```
